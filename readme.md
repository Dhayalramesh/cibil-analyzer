# 💳 AI CIBIL Score Analyzer

An AI-powered web application that analyzes CIBIL credit reports and provides financial insights.

---

## 🚀 Features

- 📂 Upload CIBIL PDF report
- 📊 Extract credit score automatically
- ⚠️ Detect financial issues (late payments, utilization, loans)
- 💡 Personalized recommendations
- 🤖 AI-powered explanation (Gemini API)
- 📈 Visual dashboard with charts

---

## 🛠️ Tech Stack

- Python
- Streamlit
- pdfplumber
- Gemini API (LLM)
- Pandas
